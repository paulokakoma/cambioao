Perguntas Frequentes

1. O cartão é físico ou virtual?

Este é um cartão 100% virtual. Receberá todos os dados (número, data de validade e código CVV) digitalmente para usar em pagamentos online.

2. Como faço para carregar o cartão?

O carregamento é feito por transferência bancária em Kwanzas (Kz). Após a solicitação do cartão, receberá todas as instruções detalhadas e os dados para o carregamento.

3. Preciso de ter uma conta bancária?

Não, não é obrigatório ter uma conta bancária para adquirir o cartão. O pagamento da taxa e os carregamentos podem ser feitos por depósito ou transferência a partir de qualquer conta.

4. Em quanto tempo recebo o cartão?

Após a confirmação do pagamento da taxa de aquisição, os dados do cartão são enviados em poucos minutos diretamente para o seu WhatsApp.
